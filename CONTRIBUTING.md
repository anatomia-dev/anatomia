# Contributing to Anatomia

The CLI contributor guide lives at [`packages/cli/CONTRIBUTING.md`](./packages/cli/CONTRIBUTING.md).

It covers:
- Adding framework detectors, gotchas, services, and skill templates
- Code patterns to avoid (type washing, phantom analyzers)
- Error handling conventions
- Branching and PR workflow

## Workspace Setup

Requires Node.js 22+ and pnpm 9+. Run `pnpm install` at the repo root.
