# Anatomia Documentation

Welcome to the Anatomia documentation. This page validates that the
Fumadocs MDX pipeline is working correctly.

## Getting Started

Install Anatomia and run your first scan:

```bash
npm install -g anatomia-cli
ana init
ana scan
```

Anatomia scans your codebase, detects your stack, and generates context
files that make AI coding agents reliable.

## How It Works

Anatomia's pipeline runs through five stages:

```typescript
// The verified development pipeline
type Stage = "scope" | "plan" | "build" | "verify" | "merge";

interface PipelineRun {
  feature: string;
  stages: Stage[];
  assertions: ContractAssertion[];
}
```

<Callout variant="rule">
  Every change must be verified. The proof chain records what was built,
  what was tested, and what was confirmed.
</Callout>

<Callout variant="note">
  This requires Node.js 18+ and a package manager (npm, pnpm, or yarn).
</Callout>
