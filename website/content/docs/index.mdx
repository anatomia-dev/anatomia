# Anatomia Documentation

Welcome to the Anatomia documentation. This page validates that the
Fumadocs MDX pipeline is working correctly.

## Getting Started

Install Anatomia and run your first scan:

```bash
npm install -g anatomia-cli
ana init
ana scan
```

Anatomia scans your codebase, detects your stack, and generates context
files that make AI coding agents reliable.
